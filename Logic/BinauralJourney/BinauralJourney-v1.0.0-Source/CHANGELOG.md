# Changelog

## 1.0.0

- Initial public source release.
- AU, VST3, and Standalone formats.
- Single-tone and Journey playback modes.
- Configurable binaural-beat stages and transitions.
- Musical-note and frequency-based anchor controls.
- Pink, brown, and white ambient noise.
- Logic transport synchronization.
- Logic automation support.
- Factory and custom presets.
- Realtime and offline bounce support.
